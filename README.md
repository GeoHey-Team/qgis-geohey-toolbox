# GeoHey-Toolbox plugin for QGIS
It a plugin of [QGIS](qgis.org) provided by [GeoHey](www.geohey.com). this tools allow user do coordinate convert when using chinese coordinate
The plugin needs QGIS 2.8 to work. 
## Tools
1. China mars coordinate covert tool  
this tools allow user do coordinate convert when using chinese coordinate.
    - WGS to GCJ02
    - GCJ02 to WGS
    - GCJ02 to BaiDu
    - BaiDu to GCJ02
    - WGS to BaiDu
    - BaiDu to WGS

2. grid tool  
this tool allow use create grid polygon.
    - equal area lambert grids generate tool
